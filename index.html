Mikeal Rogers

Director of Evangelism

Mikeal has been active in open source for more than ten years and has held prior roles at Mozilla and the Open Source Applications Foundation. An early arrival in the Node community he wrote the popular library request and created he first community driven Node conference, NodeConf. In recent years he's been a frequent conference speaker and organizer having also started JSFest, a week long JavaScript festival using entire cities as venues.

Talks

  • Essential Patterns in Node, NodeDublin 2012
  • Alchemy or Amateurization, RealtimeConf 2012
  • Performant Mobile Applications, RealtimeConf 2013
  • The Rise of Front End Tooling, Front End Ops Conf, 2014
  • The Future of Node Patterns, ScotlandJS 2014

Erika Heidi

PHP Evangelist

Erika Heidi is a web developer / devOp working with PHP and Linux for over a decade. Author of the Vagrant Cookbook and open source projects like phansible.com and dev-human.org, she worked many years as an independent developer, experimenting with a large diversity of projects and tools. Erika is an active open source contributor, speaker and writer.

Talks

  • Portable Environments with Vagrant, Laracon EU 2014
  • Vagrant Provisioners in a Nutshell, PhpDay Italy 2014
